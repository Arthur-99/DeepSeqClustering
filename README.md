# EventSeqDecomp

An **unsupervised event sequence decomposition** method is proposed to mine highly correlated subsequences (clusters) with:

- neural temporal point process 
- sequential variational autoencoder
- TGAT time encoding 
- NRI decoder 
- beta-VAE loss
